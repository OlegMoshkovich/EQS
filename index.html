This portfolio contains some of my work completed over the last 7 years.

In 2017 omission.studio was engaged to develop the concept, the vision and the design for the Girls Scouts digital experience for 2020 release.

We were a multi-disciplinary team of 4.

I was responsible for the technical aspects of the project, which included

unity and react native development.

The chosen approach was to design and organize the maximum amount of features into the mobile application in order to provide the generation Z user the wholesome experience in the form of the digital safe space ecosystem.

The product included fully customizable 3d avatar system and multi-player game world developed in unity.

Microservice architecture powered by the AWS services was chosen as the Backend system.

React Native was selected as the front end framework.

Together with Jochen Hartman we developed high fidelity React Native prototype that enabled the team to extensively test the design with 3 groups of users in

New York,

Dallas,

Washington.

User seesions validated the developmental vector, and resulted in the definition of the first release feature set.

The production team was engaged in September of 2018.

The first version of the application is on schedule to be released in March of 2019.
